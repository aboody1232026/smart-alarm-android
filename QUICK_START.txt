## 🔧 TROUBLESHOOTING QUICK FIX

| Problem | Solution |
|---------|----------|
| "Unknown app" warning | Tap "Install anyway" or enable Unknown Sources |
| APK won't install | Uninstall old version first |
| App crashes on open | Check logcat: `adb logcat \| grep smartalarm` |
| Can't connect to Tapo | Verify IP is correct, device is on WiFi |
| Can't connect to Gree | Check MAC address format (lowercase) |
| Build takes too long | First build is 30+ min - this is NORMAL |
