## ⚡ 5 خطوات فقط لإنشاء APK

### ✅ الخطوة 1: تحضير الأدوات (5 دقائق)

```bash
# تثبيت أدوات البناء
pip install buildozer cython
```

**متطلبات إضافية:**
- Java: https://www.oracle.com/java/technologies/downloads/
- Android SDK: https://developer.android.com/studio

### ✅ الخطوة 2: بناء التطبيق (15 دقيقة - أول مرة فقط)

```bash
# خيار أ: استخدام السكريبت (أسهل!)
bash build.sh
# اختر رقم 1

# خيار ب: بناء يدوي
buildozer android debug
```

### ✅ الخطوة 3: انتظر النتيجة ☕

الملف الناتج: `bin/smartalarm-1.0.0-debug.apk`

### ✅ الخطوة 4: نسخ إلى الهاتف

**الطريقة الأولى (تلقائي):**
```bash
bash build.sh
# اختر رقم 3
```

**الطريقة الثانية (يدوي):**
```bash
adb install -r bin/smartalarm-1.0.0-debug.apk
```

**الطريقة الثالثة (بدون سلك):**
- انسخ الملف إلى الهاتف عبر USB
- افتح الملف على الهاتف
- اضغط تثبيت

### ✅ الخطوة 5: استخدم التطبيق! 🎉

1. افتح Smart Alarm Pro
2. أدخل بيانات Tapo و Gree
3. ضبط المنبّه وانعم بنومك الهانئ 😴

## 🆘 مشاكل شائعة = حلول سريعة

| المشكلة | الحل |
|--------|------|
| Java غير موجود | حمّل JDK من oracle.com |
| Buildozer لا يشتغل | `pip install --upgrade buildozer` |
| خطأ في البناء | `buildozer android clean` ثم بدء من جديد |
| اتصال Tapo فشل | تأكد من IP والبريد صحيحين |
| لا يرن المنبّه | تأكد من فتح التطبيق والصوت مفعّل |
