✅ # قائمة التحقق - Smart Alarm Pro Android Build

استخدم هذه القائمة للتأكد من كل خطوة قبل البدء!

---

## 🔧 مرحلة التحضير

- [ ] تثبيت Python 3.9+
- [ ] تثبيت Java Development Kit (JDK)
- [ ] تثبيت Android SDK (أو Android Studio)
- [ ] تثبيت Buildozer: `pip install buildozer cython`
- [ ] التنقل إلى مجلد المشروع

---

## 📋 فحص الملفات

- [ ] وجود `tapos_android.py` (الملف الرئيسي)
- [ ] وجود `buildozer.spec` (إعدادات البناء)
- [ ] وجود `requirements.txt` (المكتبات)
- [ ] وجود `build.sh` (سكريبت البناء)

---

## 🔨 خطوات البناء

### المرة الأولى (قد تستغرق 20+ دقيقة):
- [ ] تنظيف البناء السابق (إن وجد)
- [ ] تشغيل: `buildozer android debug`
- [ ] الانتظار حتى الانتهاء (وضع تحميل ☕ إذا أمكن)
- [ ] عدم إغلاق Terminal أثناء البناء

### المرات اللاحقة (أسرع بكثير):
- [ ] تشغيل: `buildozer android debug`
- [ ] الانتظار حتى الانتهاء

---

## 📦 التحقق من النتائج

- [ ] الملف موجود: `bin/smartalarm-1.0.0-debug.apk`
- [ ] حجم الملف: 15-20 MB (معقول)
- [ ] عدم وجود أخطاء في النهاية
- [ ] رسالة "Finished" أو "Successfully built"

---

## 📱 التثبيت على الهاتف

### قبل التثبيت:
- [ ] توصيل الهاتف عبر USB (إن استخدمت ADB)
- [ ] تفعيل وضع المطور على الهاتف
- [ ] السماح بالتثبيت من مصادر غير معروفة

### التثبيت:
- [ ] تشغيل: `adb install -r bin/smartalarm-1.0.0-debug.apk`
- [ ] أو نقل الملف يدويّاً والنقر عليه

### بعد التثبيت:
- [ ] التطبيق يظهر في قائمة التطبيقات
- [ ] يمكن فتح التطبيق بنجاح

---

## ⚙️ الإعدادات الأولية

- [ ] فتح التطبيق
- [ ] عدم ظهور أخطاء على البدء
- [ ] ظهور 4 تبويبات رئيسية

## 🔌 اختبار الاتصالات

### Tapo P100:
- [ ] إدخال عنوان IP الصحيح
- [ ] إدخال البريد الإلكتروني
- [ ] إدخال كلمة المرور
- [ ] الضغط على "اتصال"
- [ ] ظهور رسالة "متصل ✓"
- [ ] اختبار تشغيل/إيقاف

### Gree AC:
- [ ] إدخال عنوان IP
- [ ] إدخال MAC Address
- [ ] الضغط على "اتصال"
- [ ] ظهور رسالة "متصل ✓"
- [ ] اختبار تشغيل/إيقاف

---

## 🔊 اختبار الأصوات

- [ ] الضغط على "اختبار الصوت"
- [ ] سماع صوت المنبّه
- [ ] الضغط مجدداً لإيقاف الصوت
- [ ] عدم وجود أخطاء صوتية

---

## ⏰ اختبار المنبّه

- [ ] دخول Tab "المنبّه"
- [ ] اختيار وقت قريب (مثلاً بعد دقيقة)
- [ ] الضغط على "✅ تفعيل المنبّه"
- [ ] الانتظار حتى الوقت المحدد
- [ ] تشغيل الصوت والأجهزة الذكية

### عند الرنين:
- [ ] سماع الصوت
- [ ] تشغيل Tapo (إذا كان متصل)
- [ ] إيقاف Gree (إذا كان متصل)
- [ ] ظهور خيارات "إيقاف" و"غفوة"

---

## 🧪 الاختبارات التقدمية

### اختبار الغفوة:
- [ ] تشغيل المنبّه
- [ ] الضغط على "غفوة 5 دقائق"
- [ ] الانتظار 5 دقائق
- [ ] تشغيل المنبّه مرة أخرى

### اختبار الإلغاء:
- [ ] تفعيل المنبّه
- [ ] الضغط على "إلغاء المنبّه"
- [ ] عدم رنين المنبّه في الوقت المحدد

### اختبار السجل:
- [ ] ظهور جميع الأحداث في Tab "السجل"
- [ ] عدم وجود تأخير في تسجيل الأحداث
